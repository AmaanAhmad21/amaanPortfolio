# Amaan — Portfolio

Next.js (App Router), TypeScript, Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — dev server
- `npm run build` — production build
- `npm run start` — run production build
- `npm run lint` — ESLint

## Add a favicon

Place `favicon.ico` in `public/`.
